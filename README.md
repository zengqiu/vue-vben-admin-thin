## Introduction

Vue Vben Admin Thin is the slimmed-down version of [Vue Vben Admin](https://github.com/vbenjs/vue-vben-admin).

- Removed playground.
- Removed docs.
- Remove some apps.
  - `web-antdv-next`
  - `web-ele`
  - `web-naive`
  - `web-tdesign`
- Removed some configuration files.
  - `.github`
  - `.vscode`
  - `.gitpod.yml`
  - `tea.yaml`

## Install and Use

1. Get the project code

```bash
git clone https://github.com/zengqiu/vue-vben-admin-thin.git
```

2. Install dependencies

```bash
cd vue-vben-admin
npm i -g corepack
pnpm install
```

3. Run

```bash
pnpm dev
```

4. Build

```bash
pnpm build
```
